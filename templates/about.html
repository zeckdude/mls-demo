Who I am

Unfortunately, HomeSearch.com is not a site for any actual company. It is, however, a conceptual site to demonstrate the abilities of its developer, Chris Seckler. It is built using HTML, CSS, and JavaScript. It uses an API to access demo MLS data, but can be switched over to use actual MLS data at any moment. Users can search for property listings via the map search tools, search filters, or a combination of both. It is optimized for use both on mobile and desktop environments.

If you are looking for a solution for your real estate site, I would love to create the optimal experience for your customers' needs. Please email Chris Seckler.
